# 欢迎来到 WikiMuseum

这是您的第一篇文章。WikiMuseum 是一个现代化的文章发布平台，具有以下特点：

## 主要功能

- **Markdown 支持**：使用熟悉的 Markdown 语法撰写文章
- **AI SEO 优化**：发布时自动调用 AI 优化文章的 SEO 元数据
- **自动化部署**：通过 GitHub Actions 自动构建和部署

## 如何发布文章

1. 点击右上角的「发布文章」按钮
2. 在编辑器中粘贴您的 Markdown 内容
3. 点击发布，系统会自动处理后续流程
